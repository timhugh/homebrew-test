# Timhugh Test

## How do I install these formulae?

`brew install timhugh/test/<formula>`

Or `brew tap timhugh/test` and then `brew install <formula>`.

Or, in a [`brew bundle`](https://github.com/Homebrew/homebrew-bundle) `Brewfile`:

```ruby
tap "timhugh/test"
brew "<formula>"
```
